## The Difference Between syntax errors and semantic errors
### Syntax error is when the programmer writes a program and make a mistake in the language the program expects the software to be written in, and for semantic error arises when the output expected by the user is not what they get like wanting a percentage output but instead of dividing by 100, the programmer divides by 10 (that is a semantic error)
